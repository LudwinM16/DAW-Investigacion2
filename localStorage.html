LocalStorage, ¿Que es?

Este es llamado tambien Almacenamiento local, es un tipo de almacenamiento bien conocido por programadores de JS, debido a su rapidez y eficacia. Está basado en el formato Clave-Valor, y que también este no caduca a excepción de que limpiemos el navegador.

Como implementar LocalStorage en nuestro sitio web:

Ventajas y desventajas

Estas son algunas ventajas y desventajas con las que LocalStorage cuenta:

Ventaja

-> Acceder a la información de manera rapida y eficaz.

Ventaja

-> Los datos se mantienen guardados incluso después de cerrar las ventanas.

Ventaja

-> Tiene la capacidad de compartir los valores almacenados entre las ventanas.

Desventaja

-> Cuando se acumulan muchos datos, deja de ser eficiente.

Desventaja

-> Solo es posible almacenar cadenas de texto (no booleanos, arrays, objetos, etc).

Desventaja

-> Cualquier script en la página puede tener acceso a la información.

Métodos y propiedades de LocalStorage

Al ser similar a sessionStorage, LocalStorage hereda las mismas propiedades y métodos del objeto Storage:

Propiedades



length

-> Determina la cantidad de elementos almacenados en el objeto, devolviendo dicha cantidad como un número entero.

Métodos



key()

-> Este método requiere de un número entero “n” para acceder al enésimo elemento almacenado

getItem()

-> Se debe ingresar el nombre de una clave (elemento o propiedad) para que este método devuelva el valor de dicho elemento almacenado en el Objeto

setItem()

-> Este método necesita un nombre y un valor para añadir en el Objeto una propiedad con ese nombre como su nombre o clave y cuyo valor será el ingresado, si ya existía dicha propiedad sólo actualiza su valor. Para ingresar los campos necesarios, se deben separar por comas ( , ) sessionStorage(clave, valor);

removeItem()

-> Requiere que se ingrese el nombre de una propiedad para que este método la elimine del objeto

clear()

-> Al llamar este método, se vacían el objeto Storage, eliminando todas las propiedades almacenadas y su valor o información contenida.
